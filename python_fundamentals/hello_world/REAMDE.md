# Python_fundamentals - hello_world

This project is about how **Python** executes code and to understand expression and statements, interactive execution, using `pip`, understanding difference between global and local installations and virtual environments.

## Requirements
- Correction will run on **Ubuntu 20.04 LTS**
- Python version used for correction: **Python 3.8.x**
- Every Python file must start exactly with:
```python
#!/usr/bin/env python3
```
- Every Python file must:
	- Be executable
	- End with a newline
	- Be PEP8 compliant (pycodestyle 2.14.0)
- Output must match expected formatting exactly
- No external librairies are allowed unless explicity requested

## Usage
```bash
./structured_output.py
```

## Scripts Descriptions
- [`structured_output.py`](./structured_output.py): Prints exactly
```bash
Language: Python
Version: 3
Pi approx: 3.14
Computation valid: True
```
	- The float must be derived from a numeric value and formatted to two decimals
	- The boolean must result from evaluationg a comparaison expression
	- At least one line must use formatted string interpolation
	- No input is allowed
